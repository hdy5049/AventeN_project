📌 프로젝트 개요 (Project Overview)

아반떼 N은 국산차 역사상 최고의 퍼포먼스를 보여주지만, '아반떼'라는 이름 때문에 **"그 돈이면 그랜저나 중고 G70을 산다"**는 비아냥의 대상이 되곤 합니다.

이 프로젝트는 **유튜브 댓글 빅데이터(약 10만 건)**를 수집 및 분석하여, 대중의 편견(브랜드/하차감)과 실제 오너들의 경험(재미/가성비) 차이를 데이터로 증명합니다.

특히 경쟁 차종 오너들의 훈수나 단순 비방글을 필터링하고, **실제 소유주만이 알 수 있는 경험(출고, 서킷, 고급유, 기변 등)**을 추출하여 아반떼 N의 진짜 가치를 재조명합니다.

📊 데이터 분석 시각화 (Visualization)

오너들이 아반떼 N을 선택할 때 어떤 차종과 가장 치열하게 고민했는지, 그리고 어떤 매력 포인트 때문에 최종 선택을 했는지 시각화한 결과입니다.

💡 분석 인사이트 (Insights)

최다 비교군: 국산 세단(G70)보다 BMW 3시리즈/M과의 비교 언급이 압도적으로 많았습니다. 이는 오너들이 이 차를 '이동수단'이 아닌 **'스포츠 주행 머신'**으로 인식함을 보여줍니다.

승리 요인: 경쟁차들이 '브랜드 밸류'나 '고급감'에서 우위를 점한 반면, 아반떼 N은 '운전 재미', '가성비', '코너링', '배기음' 항목에서 압도적인 지지를 받았습니다.

🏁 Epilogue : 개발자가 직접 타본 '아반떼 N' 솔직 후기

데이터 분석을 넘어, 실제 오너로서 느낀 점을 가감 없이 적어봅니다.

1. "하차감은 0점, 주행감은 100점"

솔직히 말해서 호텔 발렛 맡길 때나 친구들 만날 때 "아반떼 샀어?"라는 말 들으면 위축될 때가 있습니다. 실내 내장재요? 플라스틱 도배 맞습니다.
하지만 시동 버튼을 누르는 순간 모든 단점이 사라집니다. 주차장을 울리는 배기음과 스티어링 휠을 잡았을 때 느껴지는 노면 정보는, 이 차가 단순한 아반떼가 아님을 증명합니다.

2. "NGS 버튼은 마약이다"

핸들에 달린 빨간색 NGS(N Grin Shift) 버튼을 누르면 20초 동안 차가 짐승처럼 변합니다.
퇴근길 꽉 막힌 도로에서 스트레스받다가도, 뻥 뚫린 구간에서 NGS 한 번 쓰고 팝콘 소리(팡팡!) 들으면 회사에서의 스트레스가 순식간에 날아갑니다.
이 차는 이동 수단이 아니라, 어른들을 위한 가장 저렴하고 강력한 장난감입니다.

3. "그돈씨? 타보면 그 소리 쏙 들어갑니다"

데이터 분석 결과에서도 나왔지만, 욕하는 사람들의 90%는 안 타본 사람들입니다.
저도 구매 전엔 고민했지만, 지금은 확신합니다. 4천만 원 언더로 신차 상태에서 바로 서킷을 들어갈 수 있는 차는 전 세계에 아반떼 N 뿐입니다.
전기차 시대가 오기 전, 내연기관의 마지막 낭만을 즐기고 싶다면 지금 당장 사세요.

🛠️ 핵심 기능 및 기술 (Features & Tech)

1. 🔍 유튜브 댓글 대량 수집 (Data Collection)

Google YouTube Data API v3를 활용하여 관련 영상의 댓글을 10만 건 이상 대량 수집.

Smart Search: "아반떼N 제로백", "아반떼N vs G70", "아반떼N 뉘르부르크링" 등 이슈 키워드 자동 탐색.

2. 🛡️ 초고강도 오너 필터링 (Strict Filtering Algorithm)

이 프로젝트의 핵심 알고리즘입니다. 단순한 키워드 매칭이 아닌, '경험'의 유무를 판단합니다.
